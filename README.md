# 🐶 FriendlyPet API

API desenvolvida em C# (.NET) para mapear locais pet friendly na cidade de Pederneiras-SP.

## 🚀 Tecnologias

- .NET 9
- Entity Framework Core
- SQL Server
- Swagger

## 📌 Funcionalidades

- Cadastro de locais pet friendly
- Listagem de locais
- Busca por ID
- Atualização de dados
- Remoção de registros

## 📂 Estrutura

- Controllers → endpoints da API
- Models → entidades do sistema
- Data → contexto do banco de dados

```bash
git clone https://github.com/SEU-USUARIO/FriendlyPetAPI.git
cd FriendlyPetAPI
